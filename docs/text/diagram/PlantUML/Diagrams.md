## サンプル

```

+----------+ +----------+ +----------+ *----------* /----------\ +-----+
| Docuemnt | | Database | | I/O      | |          | |          | |     |
| {d}      | | {s}      | | {io}     | |          | |          | ++    |
+----------+ +----------+ +----------+ *----------* \----------/ ++----+
@endditaa
```
